Ruleta Activa
(Animalitos)

El popular juego de la ruleta activa con animalitos, el sistema se divide en 2 secciones (Banquero/Vendedor) dando la opción al banquero de administrar y gestionar las ventas de cada vendedor.

Comprar o Mas informacion escribir a felixablancoe@gmail.com

#Funciones del Sistema

#Banquero

  • Crea Vendedores
  • Limite de Monto de venta animalito
  • Bloque de Animalitos General/Perosnalizado
  • Actualiza perfil
  • Carga resultados de cada sorteo
  • Reportes ( Vendedor / Jugadas / Premios)

#Vendedor

  • Venta de Animalitos
  • Pago de Animalitos
  • Reportes del Vendedor
  • Estadisticas de monto de ventas del dia

#Validaciones del Juego

  • Valida todas las funciones mencionadas
  • Valida que las fichas no se vendan si se pase del monto de cada animalito estipulado por el adminstrador.
